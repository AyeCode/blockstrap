== Description ==
An experimental full site editing theme for blogs or sports teams.
This theme requires Gutenberg version 10.6 or newer.
BlockStrap includes multiple block styles and block patterns including site headers, site footers and page layouts.

Named after Diego BlockStrap Maradona https://en.wikipedia.org/wiki/Diego_Maradona
